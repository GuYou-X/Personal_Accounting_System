## ✨ 功能特点（理论上）

- ➕ 添加收入/支出记录（金额、类别、日期、备注）  
- 📋 查看所有记录列表（支持按日期排序）  
- 🗑 删除记录（手滑别怪我）  
- 📊 统计总收入和总支出（看看自己多穷）  
- 💾 数据持久化（重启不丢失，除非你删文件）

### 后端
- **Python** + **Flask**（或者 FastAPI，AI 选了个顺手的）  
- **SQLite** 数据库（轻量，适合闹着玩）
